# Brain Tumor Segmentation using U-Net

## Project Overview
This project implements a deep learning model for detecting brain tumors from MRI images using the U-Net architecture. The model performs image segmentation to identify tumor regions in MRI scans.

## Features
- Brain tumor segmentation using U-Net
- MRI image preprocessing and normalization
- Model training and evaluation
- Visualization of predicted tumor masks
- Interactive web demo using Streamlit

## Tech Stack
- Python
- PyTorch
- OpenCV
- NumPy
- Matplotlib
- Streamlit

## Model Architecture
The project uses the U-Net convolutional neural network architecture for medical image segmentation to detect tumor regions from MRI images.

## Evaluation Metrics
- Dice Coefficient
- Intersection over Union (IoU)
